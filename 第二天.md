### 2.循环
循环就是一段有规律的重复代码，用来简化书写。

- 小明跑了第1圈: `print("小明跑完了第1圈")`
- 小明跑了第2圈: `print("小明跑完了第2圈")`
- 小明跑了第3圈: `print("小明跑完了第3圈")`
- ……

#### 循环需要具备的条件
1. **初始化条件** → 循环的开始，定义一个变量 `index = 1`
2. **循环条件** → 当条件为 `True` 时执行循环，例如 `index <= 3`
3. **循环体** → 条件为 `True` 时执行的代码块
4. **迭代条件** → 记录循环次数，例如 `index += 1`

---

#### 2.1 while 循环
```python
"""
循环就是一段有规律的重复代码
while循环的语法
    index = 1           #  初始化
    while bool类型:     # 循环条件
        # 编写循环体,当循环条件为True的时候,那么就执行循环内部的代码
        # 迭代条件
        index += 1
"""
index = 1
while index <= 3:
    print(f"小明跑了第{index}圈")
    # 记录一下循环的次数
    index += 1

print("循环结束")
```

### 3.隐式类型转换
- 隐式类型转换 → 系统自动将数据类型转换为目标类型。

**示例**：
```python
"""
求 50~100 之间奇数和
"""
sum = 0
index = 50
while index <= 100:
    # 注意: 这里面就触发了一个隐式类型的转换
    # 按道理这个 index % 2 == 1 为奇数, index % 2 == 的时候就偶数
    # 看他的执行结果 if 后面的接的是bool,bool为True就执行if里面的内容,否则就不执行
    # if 后面的默认bool,那么if后面的就会根据if语法的目的自动将里面的内容转换成bool类型,自动的转换的就是隐式类型的
    # None => False
    # 字符串 => "" 完全为空的字符串
    # 数字 0 => False,其他的都是True
    if index % 2:   # index % 2 
        sum += index
    index += 1

print("程序结束",sum) 
```

### 6.Debug 调试
| 操作     | 行为                           |
| -------- | ------------------------------ |
| 继续     | 运行到下一个断点（或程序结束） |
| 单步调试 | 执行当前行，不进入函数内部     |
| 单步进入 | 执行当前行，进入函数内部       |
| 单步跳出 | 立即退出当前函数，回到调用处   |
| 切换断点 | 在当前行添加/删除断点          |

```python

步骤:
    1. 添加断点,在你要分析代码前面断点
    2. 启动调试,我们可以按 F5 来进行调试
    3. 分析代码
        Variables =>可以查看变量的变化的过程
        Console => 可以查看程序执行的结果
    4. 如果进行下一步的调试
        继续 => 直接跳转到下一个断点,如果当前断点一句是最后一个断点,那么再按一下就会直接跳到结束
        单步进入(逐过程调试) => 遇到代码,会逐句调试,遇到函数会跳转到函数内部
        单步跳过(逐行调试) => 遇到代码,会逐句调试,遇到函数也不会进入函数内部
        单步跳出(跳出函数) => 从当前函数立即执行并返回调用的地方
    5. 如果调试

    6. 添加/删除断点

    # c 大,那么就比较c 和 e 
    if c >= e:
        max = c
    else:
        max = e

    if d >= e:
        max = d
    else:
```

#### 9.4 九九乘法表表格

```python
* * * * *           第二行

规律:
    外循环控制行
    内循环控制列
分析九九乘法表
    当外循环第一行的时候,内循环最多循环一次
    当外循环第二行的时候,内循环最多循环二次

        j = 3   内循环最大为 3
                1,2,3
        ... ...
"""
for j in range(1,10):                      # 外面这个循环循环了四次
```

#### 10.3 字符串的切片
- 切片用于截取序列数据中的某一段片段。
- 特点：**包头不包尾**（左闭右开）。

**示例**：
```python
"""
切片的语法 => 字符串的变量名 [start:end:step]
    start => 开始的下标
    end => 结束的下标
    step => 步子
"""
str1 = "Hello World"
#       01234567890

print(str1[2:5])     # llo              # 从下标为2开始截取,截取到下标为5结束,没有步长,默认步长为1
print(str1[1:6:2])   # el               # 从下标为1开始截取,截取到下标为6结束,步长为2
print(str1[:5])      # Hello            # 从下标为0开始截取,截取到下标为5结束
print(str1[2:])      # llo World        # 从下标为2开始截取,到末尾结束
print(str1[:])       # Hello World      # 从开始头结束全部截取   
print(str1[::2])     # HloWrd           # 从开头到结束全部截取,步长为2
print(str1[:-2])     # Hello Wor        # 从下标为0开始截取,到下标由后面往前面数,数2位
print(str1[-4:-1])   # orl              # 从下标由后面往前面数4位开始截取,到下标由后面往前面数-1结束
print(str1[::-1])    # 翻转字符串         # 从开始到结束,步长是从后面往前面数(翻转字符串)
print(str1[::-2])    # drWolH           # 从开始到结束,步长是从后面往前面数(数2位)
```

#### 12.4 列表推导式
```python
"""
列表推导式其实就是for循环的简化方式
语法 => 
    [ 返回数据 for item in 序列数据 if 数据加工]
    得到的结果是一个列表
"""
# 将list1里面的整数丢到list2里面去
list1 = [1,3,5,7,9,2,4,6,8]
list2 = []
# 遍历这个list1,将里面的元素一个一个的取出来,判断是不是偶数,如果是偶数,丢到list2
for item in list1:
    if item % 2 == 0:
        # print(item)   
        # 如果是偶数,那么就将这个数据丢大list2里面去
        list2.append(item)
print(list2)
print("-----------使用列表推导式来实现------------")
list1 = [1,3,5,7,9,2,4,6,8]
list2 = [item for item in list1 if item % 2 == 0]
print(list2)
```
